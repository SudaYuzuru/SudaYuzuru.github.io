关于对象

什么是对象？🐘🐘or💑or ???

万物皆对象
对象是一个具体的事物，一个具体的事物就会有行为和特性。
例如：一辆车🚗，一部手机📱，一只狐狸🦊
狐狸🦊是一类事物，而正在敲这份博客的那只才是对象

特征：金色的毛，豆豆眉
行为：打哈欠，敲代码

JavaScript中的对象🤔

JavaScript中的对象其实就是生活中对对象的一个抽象
JavaScript的对象是无序属性的集合，其属性可以包含基本值、对象、数组或函数
对象就是一组没有顺序的值
我们可以把JavaScript中的对象想象成键值对，其中值可以是数据或函数
对象的行为和特征： 特征—属性，行为—方法

实例

//字面量声明对象
var obj1 = {};
//对象中的数据都是键值成对存在的
//通常来说，值是函数则称之为方法，其他类型的值都是属性
var obj2 = {age:12, name:'sho', fei:function(){}};
//实例化方式 声明对象（内置构造函数）
var obj3 = new Object();
//自定义方式 构造函数
function(){}
//实例化自定义构造函数方式 声明对象
var f = new Fun();

获取对象的属性或方法： 对象.属性名

var obj2 = {
  age: 12,
  name: 'sho',
  fei: function(){
      console.log(1);
  }
}
obj2.fei();
